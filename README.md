# 🚀  Productivity Dashboard

Productivity Dashboard is a modern and responsive productivity dashboard designed to help users organize their daily work efficiently. The application provides an intuitive interface for managing tasks, tracking goals, monitoring progress, and improving productivity through a clean and user-friendly design.

Built completely using **HTML, CSS, and JavaScript**, this project focuses on strengthening DOM manipulation skills, local storage handling, dynamic rendering, and responsive UI development.

---

## 📌 Project Overview

Managing daily tasks and goals can become difficult when everything is scattered across different applications. Productivity Dashboard solves this problem by providing a single dashboard where users can:

* Manage daily tasks.
* Track completed and pending work.
* Organize personal goals.
* View task statistics instantly.
* Switch between light and dark themes.
* Save progress automatically using local storage.

---

## ✨ Features

### 📝 Task Management

* Add new tasks instantly.
* Edit existing tasks.
* Delete unnecessary tasks.
* Mark tasks as completed or pending.
* Automatically update task status.

### 🔍 Task Filtering

* View all tasks.
* Show only pending tasks.
* Show only completed tasks.
* Quickly find important work items.

### 📊 Task Statistics

* Display total number of tasks.
* Count completed tasks.
* Count pending tasks.
* Real-time updates whenever a task changes.

### 🎯 Goal Management

* Add personal or professional goals.
* Delete completed goals.
* Keep long-term objectives separate from daily tasks.

### 🌙 Dark Mode

* One-click theme switching.
* User preference saved in local storage.
* Better experience during night usage.

### 💾 Local Storage Support

* Tasks remain saved after refreshing the page.
* Goals remain persistent between sessions.
* Theme preference is automatically restored.

### 📱 Responsive Design

* Works on desktop devices.
* Optimized for tablets.
* Mobile-friendly layout.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Saurabhmau34/Productivity Dashboard.git
```

Move into the project directory:

```bash
cd Productivity Dashboard
```

Open the project in your preferred code editor and run `index.html` in your browser.
